Your remote GitHub CMS

DashPilot is a free CMS that saves data to your own Github Repo. Just connect DashPilot to your GitHub account and instantly start editing!



Works with any Static site generator or Single Page App

DashPilot saves your data as JSON to your own Github repo, allowing you to integrate it with many Static Site Generators and Single Page Applications.



Automatically publish to GitHub Pages, Netlify or Vercel

Because your site lives on GitHub, you can automatically publish your site to GitHub Pages, Netlify, Vercel or any other service that works with GitHub.
